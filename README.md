# UniGuide AI

UniGuide AI is an AI-powered university assistant designed to help students access university information quickly and intelligently using Retrieval-Augmented Generation (RAG).

The system supports both Arabic and English queries and provides accurate answers based on official university documents and resources.

## Features

- AI-powered university assistant
- RAG-based question answering
- Arabic & English support
- Smart semantic search using vector embeddings
- FastAPI backend
- React + Vite frontend
- FAISS vector database integration
- Real-time conversational interface

## Technologies Used

- Python
- FastAPI
- React + Vite
- LangChain
- FAISS
- Sentence Transformers
- OpenAI / Gemini APIs
- Hugging Face Embeddings

## Project Architecture

The system workflow includes:
1. Document collection and preprocessing
2. Text chunking and embedding generation
3. FAISS vector indexing
4. Semantic retrieval
5. AI-generated grounded responses

## Goal

UniGuide AI aims to improve students’ access to university information through an intelligent conversational experience powered by modern NLP and RAG technologies.
